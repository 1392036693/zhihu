# mock-zhihu
仿の知乎Web。简单模仿。

## 本地部署
node: v0.12.7，mongodb: v3.0.5
> 0. git clone https://github.com/yxfanxiao/zhihu.git
  1. 启动mongodb (如果你端口不是27017，在configure.js改)
  2. cd && npm start (不需要npm install)
  3. visit: 127.0.0.1:3000
